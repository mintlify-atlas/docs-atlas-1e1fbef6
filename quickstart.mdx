---
title: "Get CC-GUI running in under five minutes"
description: "Clone the repo, install dependencies, copy the data file, and start the dev server to have CC-GUI running locally with your first Claude Code session."
---

CC-GUI needs two processes running at once: an Express + Socket.IO API server on port 3456, and a Vite dev server on port 5173. The `npm run dev` command starts both together. Follow the steps below to go from zero to your first live session.

<Note>
  CC-GUI starts with safe mode on by default. The server binds to `127.0.0.1` and a **SAFE MODE ON** banner appears in the HUD. This is the recommended posture — do not expose CC-GUI to the public internet. See [Safe mode](/configuration/safe-mode) for details.
</Note>

<Steps>
  <Step title="Clone the repository and install dependencies">
    Clone CC-GUI and install its Node.js dependencies. The install step compiles `node-pty`, which requires a C/C++ toolchain. See [Installation](/installation) if `npm install` fails.

    ```bash
    git clone https://github.com/Haroutkhac/CC-GUI.git
    cd CC-GUI
    npm install
    ```
  </Step>

  <Step title="Copy the data store">
    CC-GUI persists projects and sessions in `data/store.json`. Copy the example file to create your own local store:

    ```bash
    cp data/store.example.json data/store.json
    ```
  </Step>

  <Step title="Start the dev server">
    Run both processes with a single command:

    ```bash
    npm run dev
    ```

    This starts:
    - **API server** at `http://localhost:3456` — Express + Socket.IO, manages terminals and state
    - **Vite dev server** at `http://localhost:5173` — React frontend with hot reload

    Open **http://localhost:5173** in your browser. You should see the CC-GUI HUD with the **SAFE MODE ON** banner.
  </Step>

  <Step title="Add a project">
    Click **+ TABLE** in the top-right corner of the HUD, or press `N`. The project picker offers two sources:

    - **GITHUB** — repos from your GitHub account (requires the `gh` CLI and `gh auth login`)
    - **LOCAL** — git repos found in your home directory (scanned 3 levels deep)

    Click any repo to add it, or choose **ENTER PATH MANUALLY** to type an absolute path.
  </Step>

  <Step title="Spawn a session">
    With a project selected, press `S` to quick-spawn a session. CC-GUI starts a new `claude` process in the project's directory via a real PTY. The session card appears in the agent grid immediately.

    <Tip>
      To name the session or use a different command (e.g. `codex`, `bash`, or `npm run dev`), open the **GALLERY** view and click **+** instead.
    </Tip>
  </Step>

  <Step title="Open the terminal">
    Click any session card in the agent grid. This opens a full-screen xterm.js terminal attached to the session's PTY. Type prompts, approve tool use, and interact with Claude exactly as you would in your own shell.

    Use the carousel at the bottom to switch between sessions in the same project without closing the terminal. On mobile, swipe left or right to navigate.
  </Step>
</Steps>
